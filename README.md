# Bot Adventure Time - Telegram бот для управления мероприятиями

Telegram-бот для управления мероприятиями в частных группах с поддержкой тем (Forum Topics). Бот помогает создавать мероприятия, управлять участниками, отправлять напоминания и предоставлять актуальную информацию по событиям.

## 🎯 Основные возможности

### 📅 Создание мероприятий
- Создание мероприятий через команду `/create_event` в личных сообщениях
- Пошаговый сбор информации: название, описание, дата, время, место, стоимость, лимит участников
- Автоматическая проверка погоды через OpenWeatherMap
- Публикация форматированных сообщений в выбранных темах группы
- Поддержка карпулинга (совместных поездок)

### 👥 Управление участием
- Inline-кнопки "Пойду", "Отказаться", "В резерв"
- Динамическое обновление списков участников
- Автоматическое перемещение из резерва при освобождении мест
- Управление водителями и пассажирами для карпулинга

### ⏰ Напоминания
- Автоматические напоминания за 1 день, 3 часа, 2 часа, 1 час и 30 минут до мероприятия
- Личные сообщения участникам и сообщения в теме группы
- Восстановление напоминаний при перезапуске бота

### 📊 Дайджест мероприятий
- Еженедельный дайджест в теме "Календарь Тусовок"
- Автоматическое формирование афиши на неделю
- Команда `/digest` для ручного формирования дайджеста

## 🛠 Технологический стек

- **Язык**: Python 3.+
- **Библиотека**: aiogram 3.x (асинхронная)
- **База данных**: YDB (Yandex Database)
- **Планировщик**: APScheduler
- **Погода**: OpenWeatherMap API
- **Хостинг**: Поддержка бесплатных платформ (Render, Railway, PythonAnywhere)

## 🚀 Быстрый старт

### 1. Предварительные требования

1. **Telegram Bot Token**: Создайте бота через [@BotFather](https://t.me/BotFather)
2. **OpenWeatherMap API Key**: Получите бесплатный ключ на [openweathermap.org](https://openweathermap.org/)
3. **YDB (Yandex Database)**: Настройте базу данных в Yandex Cloud
4. **Telegram Group**: Создайте приватную группу с включенными темами

### 2. Установка зависимостей

```bash
cd bot
pip install -r requirements.txt
```

### 3. Настройка окружения

Создайте файл `.env` в директории `bot/`:

```env
BOT_TOKEN=ваш_токен_бота
WEATHER_API_KEY=ваш_ключ_openweathermap
GROUP_ID=-1001234567890  # ID вашей группы
ADMIN_IDS=123456789,987654321  # ID администраторов через запятую
TIMEZONE=Europe/Moscow

# Настройки YDB
YDB_ENDPOINT=grpcs://ydb.serverless.yandexcloud.net:2135
YDB_DATABASE=/ru-central1/ваш_путь_к_базе
YDB_SERVICE_ACCOUNT_KEY_FILE_CREDENTIALS=путь_к_ключу.json
```

### 4. Настройка бота в Telegram

1. Добавьте бота в группу
2. Назначьте бота администратором с правами:
   - Отправка сообщений в темы
   - Редактирование сообщений
   - Удаление сообщений (опционально)
3. Отключите privacy mode в @BotFather

### 5. Запуск

**Локальный запуск:**
```bash
cd bot
python main.py
```

**Запуск как веб-приложение:**
```bash
cd bot
python -m uvicorn main:app --host 0.0.0.0 --port 8000
```

## 📋 Доступные команды

### Для всех пользователей:
- `/start` - Начало работы с ботом
- `/create_event` - Создание нового мероприятия
- `/my_events` - Просмотр своих мероприятий
- `/digest` - Получить дайджест мероприятий

### Для администраторов:
- `/admin_stats` - Статистика мероприятий
- `/admin_export` - Экспорт данных

## 🔧 Конфигурация

### Настройки напоминаний
Напоминания отправляются за:
- 1 день (86400 секунд)
- 3 часа (10800 секунд)
- 2 часа (7200 секунд)
- 1 час (3600 секунд)
- 30 минут (1800 секунд)

### Настройки дайджеста
- День недели: Понедельник (1)
- Время: 10:00
